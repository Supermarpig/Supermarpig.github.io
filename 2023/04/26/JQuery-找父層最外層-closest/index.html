JQuery 找父層最外層 closest

發表於 2023-04-26 更新於 2023-04-28 分類於 學習筆記 ， web前端 閱讀次數： 1.7k 2 分鐘

前言

工作中有遇到這個情形
來記錄一下

我有一個HTML

JS中 有一段需要點及List 來找返回其他頁面
但不知道點擊的是哪一個 ID
需要到最外層 才有ID

這時候就可以用JQ 來找到

其中 e.target.closest(".Modules").id

這句話的意思就是

找到點擊的那個事件開始
往父層找 到 .Modules的那個 Class
.id然後找到他他的ID

這樣就可以了~~~
